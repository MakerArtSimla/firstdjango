About Stories, my first Django site

I have been following steps from several tutorials, but primarily this one:
  • The ultimate beginner's guide to django & python web dev website by Nick Walter, from Udemy

Other tutorials I have used include, by order of preference:

  1. The Django Project website has all the reference documentation, which is very well written, so really this should be your nr.1 stop.
  2. Writing your first Django app - the Django Project tutorial itself - very useful - has direct links to all the ref documentation (see nr. 1) that you need.
  3. Mastering Django: Core, a thorough book by Nigel George (2016). This book provides useful background information that is complementary to the practical instructions in the various tutorials. Helps you understand why things are as they are.
  4. Django Girls tutorial, nice and practical.
  5. Django Templates and Static Files by Amos Omondi from Scotch IO
  6. Ultimate Django website
  7. various discussions on stackoverflow, when troubleshooting issues
