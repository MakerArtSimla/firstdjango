    About Portfolio App

    Why

    This app was developed as a learning activity when learning to use Django framework. I was following an online course in Udemy, by Nick Walter, a very engaging presenter. He was creating a Pig Latin Translator, but I decided I wanted to provide the basis for our students to create a porfolio website to showcase their work.

    Who

    I am Benedicte, a former software developer turned educator. I design IT courses for a tertiary training institute (vocational training). Before I can teach stuff, I need to learn it...

    What

    While creating this web app, I am learning to use the following tools and technologies:

    • Django, web framework
    • Python, programming language
    • Atom, text editor
    • Git and Git Hub, version control and collaboration
    • SQLite, database management system
    • Bootstrap
